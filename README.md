# Homebrew Tap for Anvi

Terminal-native AI coding agent built in Rust.

## Install

```bash
brew tap eagleisbatman/anvi
brew install anvi
```

## Upgrade

```bash
brew upgrade anvi
```

## Install from source (HEAD)

```bash
brew install --HEAD eagleisbatman/anvi/anvi
```
